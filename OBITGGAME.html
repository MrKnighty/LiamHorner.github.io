One Boot in the Grave

Description

One Boot in the Grave is a 2d roguelite inspired by Vampire Surivors. You have to defeat increasingly difficult waves of enemies, using whatever spell combinations you collect as you progress

Project Info

This project was made as an assesment for my game development course. This project had to be loosly based on these phrases "Cycle of Life", "Shoes in places they should not be", and "Unusual Aftermath" and it had to be developed in 3 weeks only.

My Contributions

For this project, I was the solo Developer and Gameplay designer, my teammates did the art, music, and story.
